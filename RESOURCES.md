# English (IELTS 7.5) Resources

## Knowledge

### Pronunciation & connected speech (Phase 1) — British English
- [BBC Tim's Pronunciation Workshop — Consonant–Vowel Linking](https://www.youtube.com/watch?v=mV_CEIroJs8)
  The **primary source** for lesson 0001 — short, clear, British English. **Primary source for lesson 0001.**
- [BBC Learning English — Pronunciation hub](https://www.bbc.co.uk/learningenglish/features/pronunciation/)
  Gateway to *Tim's Pronunciation Workshop* and *The Sounds of English*. Default reference for the whole Phase 1.
- [The Pronunciation Lounge — Linking between vowel sounds (BBC)](https://www.bbc.co.uk/learningenglish/features/the_pronunciation_lounge/episode_14)
  BrE vowel→vowel linking (the /w/ /j/ /r/ inserts). Use for lesson 0003.
- [Rachel's English — Linking: Consonant to Vowel](https://rachelsenglish.com/english-pronunciation-linking-consonant-vowel/)
  The American-English equivalent. Kept as a fallback only if the accent ever switches to US.

### Grammar (Phase 2)
- [English Grammar in Use (Raymond Murphy, Cambridge)](https://www.cambridge.org/gb/cambridgeenglish/catalog/grammar-vocabulary-and-pronunciation/english-grammar-use-5th-edition/)
  The gold-standard self-study reference for B1–C1. **Primary source for every Phase 2 lesson.**
- [Cambridge Dictionary](https://dictionary.cambridge.org/)
  Authoritative for meaning, CEFR level, and preposition collocations. Use for every "which preposition?" question.

### IELTS (Phase 3, and always for format)
- [British Council — IELTS preparation](https://www.britishcouncil.org/exam/ielts/preparation)
  Free official skills lessons and the authoritative test-format reference.
- [IELTS Simon (Simon Corcoran)](https://www.ielts-simon.com/)
  Ex-examiner; the highest-trust free source for Writing Task 1/2 method and band-descriptor insight.
- [Cambridge IELTS authentic practice tests](https://www.cambridgeenglish.org/exams/ielts/preparation/)
  Official past papers. Finite resource — save for Phase 3 / mock phase.

### The user's trusted reference
- [English-level-up-tips (byoungd, 中文)](https://byoungd.github.io/English-level-up-tips/#/)
  A curated open-source Chinese self-study guide. Great for motivation & resource discovery; not a replacement for structured lessons.

## Wisdom (Communities)
- [r/IELTS](https://www.reddit.com/r/IELTS/)
  Active community for realistic score timelines and writing-feedback threads. Filter for ex-examiner answers.
- [r/EnglishLearning](https://www.reddit.com/r/EnglishLearning/)
  General questions and speaking-practice prompts.
- [Write & Improve (Cambridge)](https://writeandimprove.com/) — free automated IELTS-style writing feedback. Tight feedback loop for Phase 2.
- A language-exchange app (HelloTalk / Tandem) once Phase 1 linking is solid, to test skills on real humans.
