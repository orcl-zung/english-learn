Conjunctions — The Glue （连词：并列 + 从属）

Conjunctions connect clauses into sentences. Two families: coordinating (equal clauses → compound) and subordinating (unequal clauses → complex). This is the companion to Lesson 0006.

图例：黑色 = 独立主句 · 绿色 = 从句 · 连接词

① Coordinating conjunctions — FANBOYS （并列连词）

连接同等地位的成分（词、短语、或两个独立主句）。连接两个主句时，前面要加逗号。

词逻辑例句
and并列 / 补充I called her, and she answered.
but转折I called her, but she didn't answer.
or选择We leave now, or we wait till morning.
so结果 / 因此It was late, so I went home.
yet转折（出乎意料）It was raining, yet we went out.
for原因（书面/正式 ≈ because）I went to bed, for I was exhausted.
nor也不（否定后接，需倒装）He doesn't smoke, nor does he drink.

记忆：FANBOYS = For · And · Nor · But · Or · Yet · So。

Correlative conjunctions （成对的相关连词）

  • both … and — Both Tom and Jerry came.
  • either … or — Either you leave, or I will.
  • neither … nor — Neither he nor I knew.
  • not only … but also — She is not only smart but also kind.
  • not … but — It's not a bug, but a feature.

② Subordinating conjunctions （从属连词）

引导从句，并表明它和主句的关系（时间、原因、让步、条件…）。从句不能单独成句。

时间 Time

When I arrived, he had already left.

when / while / as / before / after / since / until / once / as soon as

原因 Reason / Cause

Because it was raining, we stayed in.

because / since / as / now that  （since、as 表"既然/因为"——理由已知时用）

让步 / 对比 Contrast

Although she was busy, she helped me.
He's short, whereas his brother is tall.

although / though / even though / while / whereas

条件 Condition

If it rains, we'll cancel the trip.
I'll go unless it rains.

if / unless / provided that / as long as / in case / whether … or

目的 Purpose

I left early so that I could catch the train.

so that / in order that  （后面的从句常用 can/could/may/might）

结果 Result

It was so cold that the lake froze.

so … that / such … that

地点 / 方式 Place / Manner

Sit where you like.
He looks as if he's seen a ghost.

where / wherever（地点）· as / as if / as though（方式）

Punctuation rules （逗号规则）

  • FANBOYS 连两个主句 → 逗号在 FANBOYS 前：I was tired, so I went home.
  • 从句在前 → 逗号在从句后：Because I was tired, I went home.
  • 主句在前 → 不加逗号：I went home because I was tired.

Common Chinese-learner errors （高频踩坑）

✗Although it was raining, but we went out.  中文"虽然…但是"用两个，英语只用一个
✓Although it was raining, we went out.  或 It was raining, but we went out.
✗Because I was tired, so I went home.  中文"因为…所以"用两个，英语只用一个
✓Because I was tired, I went home.  或 I was tired, so I went home.
一句话总结：中文爱成对（虽然…但是 / 因为…所以 / 除非…否则），英语每句只用一个连词。这是雅思写作里最容易被扣分的"中式英语"痕迹。
